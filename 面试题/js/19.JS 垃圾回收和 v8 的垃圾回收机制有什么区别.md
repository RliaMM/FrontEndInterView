# JS 垃圾回收和 v8 的垃圾回收机制有什么区别

## 什么是垃圾回收
js 垃圾回收是自动的，非手动释放，也叫自动内存管理。
当环境中的变量，不再使用（没有被引用的关系），则在在下一波清理过程中被回收

## V8

- 分为老生代，和新生代
  - 新生代：分为 from  to, 互相交互切换，当一个变量被交换次数达到5次以上， 晋升成为老生带，新生代速度快，容量小
  - 老生带容量大，速度慢，一些资源比较大的，会直接放在老生带中
    - 老生代回收方式：
      - 标记清除：容易产生碎片
      - 标记整理：无碎片较慢
      - 增量标记：垃圾回收与应用程序交替使用

## 内存泄露
不在使用的变量，并没有得到及时释放
