# 34.a===1&&a===2&&a===3 为 true


## 双等号 == 会触发隐式转化调用 valueOf
```js
class A {
    constructor(value) {
        this.value = value;
    }
    valueOf() {
        return this.value++;
    }
}
const a = new A(1);
if (a == 1 && a == 2 && a == 3) {
    console.log("Hi Libai!");
}

```

## 全等号，不会出发隐式转化
```js
let obj = {};
let value = 1;
Object.defineProperty(obj, 'a', {
  get() {
    return value++;
  }
})
if (obj.a === 1 && obj.a === 2 && obj.a === 3) {
    console.log("Hi Libai!");
}
```

二者并存的情况下，在数值运算中，优先调用了valueOf，字符串运算中，优先调用了toString。

## [Symbol.toPrimitive]
优先级要高于valueOf 和 toString
