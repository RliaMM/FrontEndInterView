# 模块化
多人开发中
- 命名空间冲突
- 模块复用

CommonJS
  - Node.js 是基于 CommonJS 来实现的，同步导入文件，
  - 是值的导出，后面值发生变化需要重新导入
  - 运行时加载

ESModule 是引用的导出
  - 编译时输出
  - export default 默认导出是一个值，export { a } 批量导出是一个地址

```js
let x = 10;
let y = 20;
setTimeout(() => {
  x = 100;
  y = 200
}, 10)

export  { x }
export default y
```
